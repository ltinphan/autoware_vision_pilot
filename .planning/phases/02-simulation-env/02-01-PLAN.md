## Objective

Set up Docker environment and simulation scripts for CARLA and SODA.Sim to enable real-time testing of EgoLanes perception model.

## Tasks

### Wave 1: Verify Simulation Scripts

1. **Test CARLA Launcher**
   - Run `./start_carla.sh`
   - Verify it creates output directories
   - Check it handles missing CARLA module gracefully

2. **Test SODA Launcher**
   - Run `./start_soda_sim.sh`
   - Verify it creates output directories
   - Check ROS2 environment setup

3. **Verify Docker Containers**
   - Build CARLA container: `docker build -f docker/Dockerfile.carla -t carla-sim .`
   - Build SODA container: `docker build -f docker/Dockerfile.soda -t soda-sim .`
   - Test docker-compose: `docker-compose -f docker/docker-compose.yml config`

4. **Verify CARLA Client**
   - Check Python syntax: `python3 -m py_compile simulations/carla_egolanes_client.py`
   - Verify imports resolve with proper PYTHONPATH

## Verification

Run each script and verify:
- [ ] CARLA script executes without errors

- [ ] Docker containers build successfully
